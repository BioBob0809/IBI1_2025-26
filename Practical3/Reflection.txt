Experiment Experience:
First, add the path, then use "git init" to add the ".git" file. Before "git add", you need to press "Ctrl+S" to save the modifications.
"git status" can be used to check the current branch and whether there are files saved in the index.
If you want to merge "branch1" into "master", you need to execute "git merge branch1" in the "master" branch.
"git push" can upload the local files to the GitHub website.
